==========================================================
  vocus 全文搜尋 工具　使用說明（給第一次用的人）
==========================================================

這是一個可以「用關鍵字搜尋 vocus 創作者全部文章＋留言」的小工具。
搜尋現有資料：不需要網路、不需要帳號，直接用就好。


----------------------------------------------------------
【第一步】先確認電腦有 Python（只裝一次，以後不用再裝）
----------------------------------------------------------
打開這個資料夾裡的啟動檔之後，如果它說「找不到 Python」，
就到這裡下載安裝：
    https://www.python.org/downloads/

* Windows 安裝時，請務必勾選最下面的「Add Python to PATH」再按安裝。
* Mac / Linux 通常已內建，可直接跳過。


----------------------------------------------------------
【第二步】啟動（依你的電腦選一個，雙擊它）
----------------------------------------------------------
  Windows ……… 雙擊  啟動.bat
  Mac     ……… 雙擊  啟動.command
                （若雙擊沒反應：對它按右鍵 →「打開」→ 再按一次「打開」）
  Linux   ……… 在資料夾按右鍵開終端機，輸入  ./start.sh

啟動後會自動打開瀏覽器，網址是： http://127.0.0.1:5000
（沒自動開的話，自己在瀏覽器打上面那行網址）

搜尋方式：上面打關鍵字 → 按搜尋。可選創作者、可選只搜文章/留言。
要關閉：直接把那個黑色的視窗關掉就好。


----------------------------------------------------------
【第三步】（進階）想自己「更新抓最新文章」才需要做
----------------------------------------------------------
更新需要你的 vocus 登入 cookie（因為付費內容要登入才看得到）：

  1. 開網頁後，點右上角「⚙️ 設定」
  2. 照頁面上的圖文步驟，把瀏覽器的 cookie 貼上去、按儲存
  3. 回首頁，點右上角「🔄 更新資料」，它會自動抓最新內容
  4. 跑完重整頁面就能搜到新文章

* 只想搜尋現有資料的話，這一步完全可以不用做。
* cookie 大約一天會過期，更新時若提示過期，重做步驟 1~2 即可。


有問題就問給你這個工具的人 :)
